<h1>You want to use it?</h1>

- clone the repo

- ```npm i``` 

- create a .env file and put your nomber, twilio number, SID and twilio token

```
SID =twilioSid
TOKEN=twiliotoken
TNUMBER=yourtwilionumber
NUMBER=yournumber

```

-  ```npm start```
